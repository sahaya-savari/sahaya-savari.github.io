# Asynchronous Python Guide: AsyncIO, Event Loops & Concurrency

Asynchronous programming in Python (`asyncio`) is critical for high-throughput web servers, microservices, and AI pipelines. Traditional synchronous I/O blocks the OS thread during network requests, leading to severe latency and resource exhaustion under load.

This guide details how Python's single-threaded event loop manages non-blocking I/O, Task Groups, Semaphores, and Queue worker pools.

---

## ⚙️ How the AsyncIO Event Loop Works

```mermaid
flowchart LR
    EventLoop[Event Loop] --> CallStack[Execute Coroutine to I/O Boundary]
    CallStack --> Register[Register OS File Descriptor Listener]
    Register --> NonBlocking[Yield Control Back to Event Loop]
    NonBlocking --> ReadyTasks[Check Completed I/O Events]
    ReadyTasks --> Resume[Resume Coroutine Task]
```

## 💻 Production Implementation: High-Throughput Worker Pool

```python
import asyncio
import time
from typing import List

async def worker(worker_id: int, queue: asyncio.Queue):
    while True:
        task_data = await queue.get()
        if task_data is None:
            # Shutdown signal received
            queue.task_done()
            break

        start = time.perf_counter()
        # Simulate non-blocking I/O database write
        await asyncio.sleep(0.2)
        elapsed = (time.perf_counter() - start) * 1000
        print(f"Worker-{worker_id} processed [{task_data}] in {elapsed:.1f}ms")
        queue.task_done()

async def main():
    task_queue = asyncio.Queue(maxsize=50)
    num_workers = 3

    # Launch worker tasks
    workers = [
        asyncio.create_task(worker(i, task_queue))
        for i in range(num_workers)
    ]

    # Enqueue tasks
    for item in [f"job-payload-{i}" for i in range(8)]:
        await task_queue.put(item)

    # Wait for queue items to finish processing
    await task_queue.join()

    # Send shutdown signals
    for _ in range(num_workers):
        await task_queue.put(None)

    await asyncio.gather(*workers)

if __name__ == "__main__":
    asyncio.run(main())
```
