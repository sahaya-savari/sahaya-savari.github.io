# Tailwind CSS v4 Guide: Oxide Engine, @theme Directive & Dynamic Utilities

**Tailwind CSS v4** is a complete ground-up rewrite of the utility-first CSS framework. Powered by the Rust-based **Oxide engine**, v4 compiles CSS **up to 10x faster** than v3 while replacing legacy JavaScript config files (`tailwind.config.js`) with a native, CSS-first `@theme` architecture.

This guide details configuring Tailwind v4 in Vite and React applications, defining custom design tokens, and leveraging native container queries.

---

## ⚡ Key Upgrades in Tailwind v4

| Feature | Tailwind CSS v3 | Tailwind CSS v4 (2026) |
| :--- | :--- | :--- |
| **Engine** | JavaScript / PostCSS | Rust-powered Oxide Engine |
| **Configuration** | `tailwind.config.js` | Direct CSS `@theme` directives |
| **Import Directive** | `@tailwind base; @tailwind components;` | Single `@import "tailwindcss";` |
| **Color Space** | RGB / HSL default | OKLCH high-gamut default |
| **Container Queries** | Require `@tailwindcss/container-queries` plugin | Native `@min-*` & `@max-*` container utilities |

## 💻 Code Example: CSS-First Theme Configuration in `index.css`

```css
@import "tailwindcss";

@theme {
  --font-heading: 'Instrument Serif', serif;
  --font-body: 'Barlow', sans-serif;

  --color-brand-primary: oklch(0.65 0.24 260);
  --color-brand-accent: oklch(0.78 0.18 140);
  --color-dark-bg: oklch(0.14 0.02 260);

  --radius-badge: 9999px;
  --radius-card: 1.5rem;
}

@utility liquid-glass {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.1);
}
```
