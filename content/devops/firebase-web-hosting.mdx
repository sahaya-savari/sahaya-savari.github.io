# Firebase Web Hosting, Security Rules & Cloud Functions Deployment

**Google Firebase** offers a production-grade infrastructure platform for web applications, combining global CDN hosting, real-time Firestore database synchronization, and serverless Node.js / Python Cloud Functions.

This guide details configuring `firebase.json`, setting up granular Firestore security rules, and deploying automated GitHub Actions previews.

---

## 💻 Configuration Manifest Example (`firebase.json`)

```json
{
  "hosting": {
    "public": "dist",
    "ignore": [
      "firebase.json",
      "**/.*",
      "**/node_modules/**"
    ],
    "rewrites": [
      {
        "source": "**",
        "destination": "/index.html"
      }
    ],
    "headers": [
      {
        "source": "**/*.@(jpg|jpeg|gif|png|webp|svg|woff2)",
        "headers": [
          {
            "key": "Cache-Control",
            "value": "max-age=31536000, immutable"
          }
        ]
      }
    ]
  }
}
```
